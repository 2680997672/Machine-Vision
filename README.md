# Machine-Vision
本仓库为《机器视觉与图像处理》课程大作业，选题方向 3：单数字图像简易 OCR 识别系统，基于 MATLAB + 图像处理工具箱开发。核心创新为待测图像与 0-9 数字模板采用同源全流程预处理，统一灰度化、高斯滤波、自适应二值化、形态学开运算、连通域裁剪、32×32 等比例归一化全套参数，消除传统模板匹配系统误差。 程序实现图片交互式读取、全流程可视化输出、归一化互相关匹配打分，自动定位画面内单个印刷数字并输出识别结果与匹配置信度。适配 jpg/png/bmp 图像格式，通过连通域面积筛选剔除背景干扰，等比例居中缩放消除拍摄尺度差异。 仓库包含完整可运行源代码、0~9 标准数字模板素材、完整实验报告、README 运行说明。系统轻量化、无需训练数据集，适合固定场景单数字识别。
